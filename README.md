## 🚀 快速开始

### 环境要求

- JDK 25+
- PostgreSQL
- Maven 3.9+

### 启动步骤

1. 配置数据库连接
2. 运行SQL脚本初始化数据库
3. 启动应用: `./mvnw spring-boot:run`

## 🔧 技术栈

- Spring Boot 3.5
- MyBatis-Plus 3.5
- PostgreSQL
- Sa-Token 认证
- SpringDoc OpenAPI 3
